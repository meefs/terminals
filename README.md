# terminals

A field report on the state of **terminal rendering in 2026** — what replaced `libcaca`/`aalib`,
the **Kitty / Sixel** graphics protocols, the Unicode **octant** character ladder, the **npm + TUI-framework**
renaissance (Ink · Bubble Tea · Ratatui · Textual), and opinionated guidance on what to actually use.

**▶ Live:** https://meefs.github.io/terminals/

## What it is
A single, self-contained `index.html` based on the **MARPAT** design
system by [meefs](https://github.com/meefs) : a pixel-camo background, a frosted-glass content panel, and editorial typography (Fraunces +
Newsreader + IBM Plex Mono). Controls in the top-right (persisted to `localStorage`):

- **Theme** — Navy / Army / Arctic
- **Text size** — A / A / A
- **Pixel scale** — Macro / Standard / Micro camo cells
